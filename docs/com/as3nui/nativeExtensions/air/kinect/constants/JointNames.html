Classpublic class JointNames
InheritanceJointNames Object

Names of joints, which are available on both platform.

You can use these names, together with the user class:var leftHand:SkeletonJoint = user.getJointByName(JointNames.LEFT_HAND);

A user can have more than these joints, depending on the platform. You can find all joints in the user.skeletonJoints vector.
